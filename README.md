# grasp
- android app to track things

# basic idea
- track everything
 - sport (time, iterations, intensity)
 - series progress
 - others...
- make trackings groupable
- store only on local device
- provide export and import of data
- use apis to collect/autocomplete data(imdb)

# examples
- group: sport
 - squat max
  - date: 01.01.2001
  - amount: 56x
  - execution: hard
 - aphrodite
  - date: 02.01.2021
  - duration: 46,23
  - execution: easy
- group: series
 - game of thrones
  - season: 1
  - episode: 1
  - progess: 100%
..
